# 使用[Github](https://github.com)
## 目的
借助*github*托管项目代码
## 基本概念
**仓库(Repository)**  
用来存放项目代码，每个项目对应一个仓库，多个开源项目对应多个仓库  
 **收藏(Star)**  
 **复制克隆项目(Fork)**  
 ps:该fork的项目是独立存在的  
 **发起请求(Pull Request)**  
 **关注(Watch)**  
 **事物卡片(Lssue)**
